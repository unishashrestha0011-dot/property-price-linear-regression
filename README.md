# Multiple Linear Regression using Property Dataset

## Overview
This project implements Multiple Linear Regression models to predict property prices using property characteristics.

## Features Used
- Square_Footage
- Num_Bedrooms
- Num_Bathrooms
- Year_Built
- Lot_Size

## Target Variable
- Price

## Models Implemented
1. Square_Footage + Num_Bedrooms
2. Square_Footage + Num_Bedrooms + Num_Bathrooms
3. Square_Footage + Num_Bedrooms + Num_Bathrooms + Year_Built
4. Square_Footage + Num_Bedrooms + Num_Bathrooms + Year_Built + Lot_Size

## Evaluation Metric
- R-Squared (R²)

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab
